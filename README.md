"闪紫"（英文名shiningZ）社工字典生成工具，支持ABC自定义排列组合、ABC列字典AI扩展、AI提示词联想字典、自定义AI提示、字典去重、结果导出等

`目前公开更新版本为v1.1及其小更新版本例如v1.1.1，源码已发布`
